## 📖 About the Project

This is the personal portfolio of **Riyad El Kihel**, an engineering student in Computer Science & Artificial Intelligence at ENSA Safi, Morocco.

The website showcases academic projects, technical skills, professional certifications, and provides a direct way to get in touch — all wrapped in a polished, glassmorphism-inspired interface with smooth animations and full dark/light theme support.

---

## ✨ Features

| Category | Details |
|---|---|
| **🎨 Design** | Glassmorphism UI · Premium typography (Inter, Playfair Display, Space Mono) · Subtle grid background pattern |
| **🌗 Theming** | Dark & light mode toggle with `localStorage` persistence |
| **💫 Animations** | Scroll-triggered reveal animations powered by Framer Motion |
| **📱 Responsive** | Fully responsive across mobile, tablet, and desktop breakpoints |
| **🧭 Navigation** | Floating glassmorphic navbar · Smooth scroll · Active section tracking via `IntersectionObserver` · Mobile slide-down menu |
| **📂 Projects** | Dynamic project cards rendered from data · Tech stack badges · "Coming soon" placeholders |
| **📬 Contact** | Working contact form integrated with Formspree · Country-selector phone input with search |
| **📄 CV** | One-click CV/resume download |
| **🔍 SEO** | Descriptive meta tags and semantic HTML structure |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) ≥ 18
- npm (bundled with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/elkihelriyad/My-personal-portfolio.git

# Navigate into the project
cd My-personal-portfolio

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

The dev server will start at `http://localhost:5173` with hot module replacement.

---

## 📦 Build & Deployment

### Production Build

```bash
npm run build
```

The optimised output will be generated in the `dist/` directory.

### Deploy to GitHub Pages

The project includes `gh-pages` as a dependency. To deploy:

```bash
npx gh-pages -d dist
```
